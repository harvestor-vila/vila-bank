part 1: the question stem
The visualization above is only a demonstration of the relevant data. Which visualization design would be most appropriate for comparing the total number of innovations across different categories over the years?

part 2: the option set that includes the correct answer and three incorrect options
(A) A pie chart showing the proportion of each category's total value across all years.
(B) A stacked area chart displaying the cumulative values of innovations for each category over time.
(C) A scatter plot with individual data points representing each year's total innovations per category.
(D) A single bar chart with bars representing the sum of innovations for each category across all years.

part 3: the correct answer
(B) A stacked area chart displaying the cumulative values of innovations for each category over time.
