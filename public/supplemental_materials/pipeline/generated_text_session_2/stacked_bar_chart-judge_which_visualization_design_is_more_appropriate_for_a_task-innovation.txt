part 1: the question stem
The visualization above is only a demonstration of the relevant data. Which visualization design would best allow a viewer to compare the total number of innovations across different sectors?

part 2: the option set that includes the correct answer and three incorrect options
(A) A pie chart with different slices representing each sector's total number of innovations.
(B) A stacked bar chart with different colors representing various types of innovations within each sector.
(C) A line graph with time on the x-axis and total number of innovations on the y-axis, with separate lines for each sector.
(D) A sorted bar chart with each bar representing a sector's total number of innovations, ordered from highest to lowest.

part 3: the correct answer
(D) A sorted bar chart with each bar representing a sector's total number of innovations, ordered from highest to lowest.
