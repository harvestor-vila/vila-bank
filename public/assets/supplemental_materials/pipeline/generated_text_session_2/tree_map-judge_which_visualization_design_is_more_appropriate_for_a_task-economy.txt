part 1: the question stem
The visualization above is only a demonstration of the relevant data. Which visualization design would be most appropriate for comparing the relative sizes of the subcategories within each category?

part 2: the option set that includes the correct answer and three incorrect options
(A) A pie chart with different colors representing each subcategory.
(B) A line graph with time on the x-axis and percentage on the y-axis, with each subcategory represented by a different line.
(C) A stacked bar chart with each category on the x-axis and percentage on the y-axis, with subcategories stacked within each bar.
(D) A scatter plot with subcategories on the x-axis and percentage on the y-axis, with different colors for each category.

part 3: the correct answer
(C) A stacked bar chart with each category on the x-axis and percentage on the y-axis, with subcategories stacked within each bar.
