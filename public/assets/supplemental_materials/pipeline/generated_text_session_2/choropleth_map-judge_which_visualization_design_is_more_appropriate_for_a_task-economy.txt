part 1: the question stem
Given the task of comparing economic values across different states, which visualization design would be most appropriate? The visualization above is only a demonstration of the relevant data.

part 2: the option set that includes the correct answer and three incorrect options
(A) A pie chart with slices representing the economic values of each state.
(B) A choropleth map with states colored according to their economic values.
(C) A line graph showing the economic values of each state over time.
(D) A bar chart with alphabetical labels for each state and bars representing average temperatures.

part 3: the correct answer
(B) A choropleth map with states colored according to their economic values.
